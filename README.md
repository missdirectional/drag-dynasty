## 🛠️ Tech Stack & Architecture
* **Frontend:** Snack (Mobile App) & Netlify (Web Deployment)
* **Backend & Database:** Supabase
* **Media & Video Processing:** Mux
* **Network & DNS:** Cloudflare
* **Community Waitlist:** Jotform
